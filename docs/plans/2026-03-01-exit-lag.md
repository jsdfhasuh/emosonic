### Task 2: Cache close behavior to avoid prefs read

**Files:**
- Modify: `lib/main.dart:161-175`
- Modify: `lib/main.dart:234-269`

**Step 1: Cache behavior on init**

```dart
String _windowCloseBehavior = 'ask';

Future<void> _loadWindowCloseBehavior() async {
  final prefs = await SharedPreferences.getInstance();
  _windowCloseBehavior = prefs.getString('window_close_behavior') ?? 'ask';
}
```

Call `_loadWindowCloseBehavior()` in `initState()`.

**Step 2: Use cached behavior in onWindowClose**

```dart
final behavior = _windowCloseBehavior;
```

**Step 3: Verify**

Run: `flutter run`

Expected: close no longer blocks on prefs read.
