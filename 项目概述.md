# EmoSonic 项目概述

## 📱 项目介绍

**EmoSonic** 是一个使用 Flutter 构建的跨平台 Subsonic 音乐播放器。该项目支持 Android 和 Windows 平台，旨在为用户提供高质量的音乐播放体验。

### 项目定位
- **类型**：跨平台音乐播放应用
- **技术栈**：Flutter 3.24.0+
- **支持平台**：Android、Windows
- **目标用户**：Subsonic/Navidrome 等自建音乐服务器用户

---

## ✨ 核心功能

| 功能模块 | 说明 |
|---------|------|
| **Subsonic API 支持** | 支持连接任何兼容 Subsonic 的服务器（Navidrome、Airsonic 等） |
| **后台播放** | 应用最小化或切换到后台时继续播放音乐 |
| **媒体控制集成** | Windows 上集成 SMTC，Android 上集成 MediaSession |
| **歌单管理** | 创建、编辑、管理个人歌单 |
| **离线缓存** | 缓存专辑封面和元数据信息 |
| **自动 Scrobbling** | 自动追踪和记录播放历史 |

## 🚀 快速开始

### 环境要求
```bash
# 检查 Flutter 版本
flutter --version

# 必须 >= 3.24.0
# Dart SDK >= 3.0.0
```

### 开发步骤

1. **获取依赖**
   ```bash
   flutter pub get
   ```

2. **运行应用**
   ```bash
   # Android
   flutter run -d <android_device_id>
   
   # Windows
   flutter run -d windows
   ```

3. **构建发布版本**
   ```bash
   # Android
   flutter build apk --release
   flutter build appbundle --release
   
   # Windows
   flutter build windows --release
   ```

## 📦 安装使用

### Android 用户
1. 从 [GitHub Releases](../../releases) 下载 `app-release.apk`
2. 在设置 → 安全中启用"安装未知来源的应用"
3. 安装 APK
4. 打开应用并配置 Subsonic 服务器

### Windows 用户
1. 从 [GitHub Releases](../../releases) 下载 `emosonic_windows_v*.zip`
2. 解压 ZIP 文件到指定位置
3. 运行 `emosonic.exe`
4. 首次启动时配置 Subsonic 服务器

## 📋 项目架构

### MVC 架构模式
```
User Interface Layer (UI)
        ↓
    Providers (State Management)
        ↓
    Services (Business Logic)
        ↓
    Data Layer (Repositories & Models)
        ↓
    Subsonic API / Local Storage
```

### 关键模块说明

| 模块 | 功能 |
|-----|------|
| **UI 层** | Flutter Widget，负责用户交互和界面展示 |
| **Providers** | 使用 Riverpod 管理应用状态 |
| **Services** | 业务逻辑处理，如播放、认证等 |
| **Data 层** | 数据模型、API 调用、本地存储 |
| **Core** | 常量、主题、工具函数等通用功能 |
